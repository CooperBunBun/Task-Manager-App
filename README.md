# Task Manager App

A full-stack task manager application built with HTML, CSS, JavaScript, Node.js, and Express.

## Features
- View all tasks
- Add new tasks
- Mark tasks as complete
- Delete tasks

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Other: CORS, REST API

## How to Run

### Backend
1. Open terminal in the `backend` folder
2. Run:
   ```bash
   npm install
   node server.js
